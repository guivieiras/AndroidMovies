# AndroidMovies
## Como rodar
1. Dar checkout na branch dev
2. Abrir a pasta do projeto com o Android Studio
3. Esperar o Gradle baixar as dependências
4. Executar o projeto via emulador ou smartphone Android versão 8.0 ou maior (Oreo)
